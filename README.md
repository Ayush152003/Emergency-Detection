# Multimodal Emergency Detector
A unique AI system for detecting emergencies using vision, text, and sensors.

## Features
- Multimodal fusion for high accuracy.
- Real-time API with alerts.

## Setup
1. Clone: `git clone https://github.com/yourusername/Multimodal-Emergency-Detector.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python code/emergency_detector.py`

## Usage
- Train model or use API for detection.
- Demo: See PPT in /docs.
